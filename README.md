# Airline Database Management System

A web-based Airline Database Management System built using **ASP.NET Core Razor Pages**.  
This application allows administrators to manage airline operations including customers, flights, reservations, airports, payments, and more.

## Technologies Used

- ASP.NET Core Razor Pages
- C#
- SQL Server
- Bootstrap
- HTML / CSS / JavaScript

## Features

- Admin Login Authentication
- Customer Management
- Flight Management
- Aircraft Management
- Reservation Handling
- Airport and Gate Management
- Payment Processing
- Luggage Tracking
- Airline Staff Management
- Special Services Management
